# AI Horror Story Website

## Folder Structure
- frontend/ → React frontend
- backend/ → Express backend + AI integration
- .env → API keys

## How to Run Locally
1. cd backend
2. npm install
3. node app.js
4. cd ../frontend
5. npm install
6. npm run dev

## Environment Variables
- OPENAI_API_KEY
- ELEVENLABS_API_KEY
- RUNWAY_API_KEY
